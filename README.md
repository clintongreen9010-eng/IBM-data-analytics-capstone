# IBM Data Analyst Professional Certificate Capstone

## Overview

This capstone project was completed as part of the IBM Data Analyst Professional Certificate.

The project focused on analyzing business data, identifying trends, creating visualizations, and communicating findings through an interactive IBM Cognos dashboard and presentation.

---

## Tools Used

- Python
- SQL
- IBM Cognos Analytics
- Excel
- Data Visualization
- Exploratory Data Analysis (EDA)

---

## Project Workflow

1. Explored the provided business dataset
2. Investigated key metrics and trends
3. Created visualizations to communicate findings
4. Designed an interactive dashboard in IBM Cognos
5. Presented insights and business recommendations
